🧩 Reference

Sections

PawPantry ships with 13 sections — 3 global and 10 page-specific. All are managed via Online Store → Themes → Customise.

ℹ️

Global sections (header, footer, announcement bar) appear on every page and are configured in Theme settings. Page sections are tied to specific templates and can be reordered or hidden per page in the editor.

Global sections

Global

Header header.liquid

The site-wide navigation bar rendered on every page. Includes the logo, main navigation menu, search, wishlist, account, and cart icon with item count.

  • Supports text and image logos (configured in Header settings).
  • Optional sticky scroll behaviour.
  • On mobile, navigation collapses into a full-screen drawer.
Global

Footer footer.liquid

Site-wide footer with brand copy, contact information, navigation links, and social media icons. All content is controlled via Footer settings — no code required.

Global

Announcement bar announcement-bar.liquid

A slim, dismissible banner above the header. Supports plain text with an optional clickable link. Controlled via Announcement bar settings.

Homepage sections

These sections are available on the homepage (templates/index.json). They can be reordered by dragging in the theme editor, and hidden individually by toggling the eye icon.

Homepage

Hero hero.liquid

A full-width banner at the top of the homepage. Displays a headline, subheading, and call-to-action button over a background image.

  • Recommended image size: 1440 × 700 px or larger.
  • Use high-contrast imagery so overlaid text remains readable.
Homepage

Benefits bar benefits-bar.liquid

A horizontal strip of 3–4 trust signals below the hero — for example "Free UK delivery over £40", "Vet-approved recipes", "No nasty additives". Each benefit shows an icon and short label.

Homepage

Featured products featured-products.liquid

A curated product grid (typically 4–8 products) using the product-card snippet for consistent layout. Editors choose which collection or individual products to feature.

Homepage

Categories grid categories-grid.liquid

A visual grid linking to product collections — Dry food, Wet food, Treats, Supplements, etc. Each cell shows a collection image and name.

Homepage

Testimonials testimonials.liquid

A carousel or grid of customer review quotes with star ratings, reviewer name, and optional product link. Content is managed directly in the section editor — no third-party review app required.

Page-specific sections

Product

Product page main-product.liquid

The core product display section. Rendered on every product page.

  • Product image gallery with zoom on hover
  • Title, price, and compare-at price
  • Variant selector (size, flavour, etc.)
  • Quantity picker and Add to basket button
  • Wishlist toggle
  • Accordion tabs: Description, Ingredients, Nutritional info
  • Customer review section with star rating summary
Collection

Collection page main-collection.liquid

The product listing page. Rendered on all collection pages.

  • Sort-by dropdown: Featured, Price low–high, Price high–low, Newest, Best selling
  • Filter sidebar: Category, Dog age, Price range
  • Responsive product grid using the product-card snippet
  • Pagination via the pagination snippet
Cart

Cart page main-cart.liquid

  • Line item list with image, name, variant, quantity picker, and remove button
  • Free delivery progress bar (driven by Cart settings)
  • Order summary with subtotal and delivery charge
  • Proceed to checkout button
Contact

Contact page main-contact.liquid

A contact form with name, email, and message fields. Submissions are routed through Shopify's native contact form handler. A success message is shown after submission without a page reload.

Search

Search results main-search.liquid

Displays results for products, pages, and articles. Shows the query, result count, and a grid of matched products using the product-card snippet.

Snippets

Reusable template partials included by sections. They can't be added directly to a page, but their output appears wherever the parent section is used.

File Purpose Used by
product-card.liquid Renders a single product card with image, title, price, quick-add, and wishlist icon featured-products, main-collection, main-search
pagination.liquid Renders numbered pagination controls main-collection, main-search
breadcrumb.liquid Renders a breadcrumb trail: Home › Collection › Product main-product, main-collection
